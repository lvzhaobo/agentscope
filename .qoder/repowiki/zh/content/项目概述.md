## 项目简介

AgentScope 是一个面向多智能体系统的开发框架，旨在为构建基于大语言模型（LLM）的复杂应用提供灵活且稳健的平台。该项目以开发者为中心，强调透明性、模块化和可定制性，支持从初学者到专家级开发者的不同需求。AgentScope 通过提供一系列核心模块，如智能体编排、工具调用、记忆管理、实时干预等，简化了多智能体系统的开发流程。

项目当前版本为 v1.0.10，基于 Python 3.10+ 构建，采用 Apache License 2.0 开源协议。其核心设计原则是“对开发者透明”，确保提示工程、API 调用、智能体构建和工作流编排等所有环节都对开发者可见且可控，避免深度封装或隐式魔法。

## 核心定位与设计理念

AgentScope 的核心定位是成为一个“乐高式”的多智能体系统开发框架。其设计理念围绕以下几个关键点展开：

- **对开发者透明**：这是 AgentScope 的首要原则。所有开发过程，包括提示工程、API 调用、智能体构建和工作流编排，都保持对开发者可见和可控，拒绝任何形式的深度封装或隐式行为。
- **模型无关性**：开发者可以编写一次代码，然后在不同的大语言模型上运行，无需为每个模型重写逻辑，极大地提高了开发效率和灵活性。
- **高度可定制化**：框架鼓励开发者对工具、提示、智能体、工作流以及第三方库和可视化组件进行深度定制，以满足特定的应用需求。
- **面向多智能体**：专为多智能体系统设计，支持显式的消息传递和工作流编排，确保系统内部的交互清晰明了。
- **实时介入**：原生支持实时中断和自定义处理，允许开发者在智能体运行过程中进行干预，将中断转换为智能体可观察的事件，从而实现无缝恢复。

## 技术愿景与适用场景

AgentScope 的技术愿景是推动“面向智能体的编程”（Agent-Oriented Programming, AOP）成为下一代 LLM 应用开发的新范式。其短期目标是提供一个部署友好、支持二次开发的综合性平台。

### 适用场景

AgentScope 适用于多种复杂应用场景，包括但不限于：

- **自动化任务处理**：通过 ReAct 智能体和工具调用，自动执行如代码执行、文件操作、网络请求等任务。
- **多角色协作系统**：利用 `MsgHub` 和管道（pipeline）构建多智能体对话系统，实现不同角色智能体之间的协作与辩论。
- **智能信息检索（RAG）**：集成检索增强生成（RAG）功能，支持从文本、PDF 等多种格式的文档中检索相关信息，为智能体提供知识支持。
- **长期记忆管理**：通过集成 Mem0 和 ReMe 等技术，为智能体提供长期记忆能力，使其能够记住历史交互和用户偏好。
- **语音交互应用**：支持 TTS（文本转语音）和语音识别，构建能够进行语音对话的智能体。

## 关键能力详解

### 智能体编排与工作流

AgentScope 提供了 `MsgHub` 和多种管道（如 `sequential_pipeline`）来简化多智能体对话的构建。`MsgHub` 充当消息中心，管理多个智能体之间的消息路由和信息共享，支持动态添加或删除参与者。

```python
async with MsgHub(participants=[agent1, agent2], announcement=msg) as hub:
    await sequential_pipeline([agent1, agent2])
```

### 工具调用与 MCP

框架支持同步和异步工具函数，并可通过 MCP（Model Calling Protocol）客户端将外部服务作为本地可调用函数使用。开发者可以精细控制 MCP 客户端和函数级别。

### 记忆与状态管理

AgentScope 提供了短期记忆（`InMemoryMemory`）和长期记忆（`LongTermMemoryBase`）支持。会话（`Session`）模块提供应用级别的自动状态管理，确保状态在不同执行间保持一致。

### 实时干预与追踪

`ReActAgent` 原生支持实时中断和自定义处理。同时，框架基于 OpenTelemetry 提供了全面的追踪功能，可连接到阿里云云监控、Arize-Phoenix、Langfuse 等第三方平台。

## 发展历程与未来规划

### 发展历程

- **2025-08**：AgentScope v1 正式发布，全面拥抱异步执行，带来多项新功能和改进。
- **2025-09**：上线 RAG 模块、计划（Plan）模块，开源 AgentScope Runtime 和 AgentScope Studio。
- **2025-11**：集成 ReMe 增强长期记忆，支持 Anthropic Agent Skill 和 Agentic RL。
- **2025-12**：支持 A2A（Agent-to-Agent）通信协议和 TTS 模型。

### 未来规划

根据路线图，AgentScope 的长期目标是确立“面向智能体的编程”为新的编程范式。短期规划包括：
- 提供更强大的部署能力，如沙盒化工具执行。
- 增强 AgentScope Studio，提供可视化开发、追踪和调试组件。
- 提供内置的智能体评测工具包，支持结果可视化。

## 学习路径指引

### 初学者路径

1.  **安装与配置**：通过 `pip install agentscope` 或从源码安装。
2.  **快速入门**：运行 `Hello AgentScope!` 示例，创建一个简单的 ReAct 智能体。
3.  **核心概念**：学习消息（Message）、智能体（Agent）、模型（Model）等基本概念。
4.  **功能探索**：尝试使用内置工具（如执行 Python 代码）和简单的记忆管理。

### 进阶开发者路径

1.  **多智能体系统**：学习使用 `MsgHub` 和管道构建复杂的多智能体对话。
2.  **高级功能**：深入研究 MCP 集成、RAG、长期记忆和实时干预。
3.  **性能与追踪**：利用 OpenTelemetry 追踪功能优化应用性能。
4.  **贡献与扩展**：阅读贡献指南，为项目贡献代码或开发新的功能模块。
